# dev-security-notes

Repositorio personal de apuntes técnicos, diagramas y recursos de estudio en el camino hacia convertirme en **desarrollador backend especializado en seguridad web**.

Este repositorio es mi "segundo cerebro": aquí documento lo que aprendo antes de aplicarlo en código (ver [`code-labs`](https://github.com/Caleb-Acarapi-Dev/code-labs) para los experimentos prácticos, y mis repos de portafolio para los proyectos finales).

---

## 📚 Contenido

| Carpeta | Descripción |
|---|---|
| `obsidian/backend/` | Apuntes de Python, FastAPI, Django, PostgreSQL, SQLAlchemy |
| `obsidian/frontend/` | Apuntes de Vue.js (Composition API, Pinia) y TailwindCSS |
| `obsidian/security/` | OWASP Top 10, Burp Suite, metodología de bug bounty |
| `obsidian/certifications/` | Notas de preparación para eJPT y PCEP/PCAP |
| `excalidraw/diagrams/` | Diagramas de arquitectura, flujos de autenticación (JWT/OAuth2), modelos RBAC, esquemas de base de datos |

---

## 🎯 Objetivo de este repositorio

- Consolidar el aprendizaje mediante documentación propia (no solo consumir contenido, sino re-explicarlo con mis palabras)
- Tener una referencia rápida y searchable para cuando retome un concepto meses después
- Servir como evidencia de proceso de aprendizaje activo, no solo de resultados finales

---

## 🗺️ Ruta de estudio actual

```
Fase 1 — Bases sólidas          → Python intermedio, FastAPI, PostgreSQL/SQLAlchemy, Docker
Fase 2 — Seguridad web          → OWASP Top 10, JWT/OAuth2, RBAC, Burp Suite
Fase 3 — Django + Frontend      → Django/DRF, Vue.js, TailwindCSS
Fase 4 — Portafolio + certs     → Proyectos finales, eJPT, PCEP/PCAP, bug bounty activo
```
